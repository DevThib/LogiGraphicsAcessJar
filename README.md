# LogiGraphics logiciel
Vous trouver ici les 2 fichiers nécessaires au lancement du logiciel.

Ce repository est mit à jour lorsqu'une mise à jour terminée arrive sur LogiGraphics.

Pour lancer le logiciel,il suffit simplement de lancer le start.bat et c'est parti !
 
LogiGraphics est un logiciel réalisé en javafx qui permet de dessiner des formes élémentaires tel que des rectangles,cercles,lignes afin de créer des plans ou des schémas.
Les éléments peuvent être découpés avec d'autres éléments,personnalisés en modifiant leur taille,leur position X et Y etc...
